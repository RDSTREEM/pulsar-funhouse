## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+ recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation
```bash
npm install
```

### Running Locally
```bash
npm run dev
```
Visit [http://localhost:3000](http://localhost:3000) in your browser.
